Ejemplos de Alineación de Texto

Alineación a la Izquierda - left-align

Este texto está alineado a la izquierda. La alineación predeterminada para el texto en la mayoría de los elementos de bloque.

Alineación a la Derecha - right-align

Este texto está alineado a la derecha.

Alineación al Centro - center-align

Este texto está centrado. Se usa comúnmente para encabezados y elementos de texto cortos.

Alineación Justificada - justify-align

Este texto está justificado. El texto se extenderá para llenar el contenedor, con un espaciado uniforme entre las palabras.

Alineación Justificada Completa - justify-all-align

Este texto está justificado hasta los bordes del contenedor. La última línea también está justificada al ancho completo.

Alineación de Inicio - start-align

Este texto está alineado al inicio del contenedor.

Alineación de Fin - end-align

Este texto está alineado al final del contenedor.

Alineación Inicial - initial-align

Este texto está alineado con el valor inicial, valor heredado de su elemento padre.

Alineación Heredada inherit-align

Este texto hereda la alineación de su elemento padre. Esto será el mismo que la alineación establecida en el elemento padre.

Alineación Desestablecida - unset-align

Este texto está alineado con el valor desestablecido. Esto elimina cualquier alineación previamente establecida y recae en el comportamiento predeterminado, que es similar a `initial`.
